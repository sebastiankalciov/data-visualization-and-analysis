# data-visualization-and-analysis
Repository to store the project for Data Visualization and Analysis class.


In `app/` is stored the web app which is deployed on Vercel.

In `notebook/` is stored:
- the dataset used: `notebook/dataset`
- Jupyter Notebook already compiled: `DVA_Project.ipynb`
- generated nodes data used for network graph: `nodes.json`
- generated link data used for network graph: `links.json`

Link to Web App: [here](https://data-visualization-and-analysis.vercel.app/)
